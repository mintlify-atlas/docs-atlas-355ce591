Antigravity CLI exposes a set of slash commands that you can run directly from the terminal interface. Type `/` at any prompt to access them. Each command performs a specific action — from resuming a previous session, to signing out, to inspecting your current model quota in real time.

## Quick reference

| Command | Category | Description |
| :--- | :--- | :--- |
| `/resume` | Session | List and resume a previous session; press `ctrl+delete` to delete a session from the list |
| `/logout` | Account | Clear saved credentials from the system keyring and sign out |
| `/settings` | Settings | Open the settings panel and color scheme selection |
| `/usage` | Quota | View real-time model usage statistics with a forced reload of model configuration and quotas |
| `/quota` | Quota | View remaining quota and consumption data with a forced real-time reload |
