---
title: "Antigravity CLI integrations with Antigravity 2.0"
description: "Antigravity CLI and Antigravity 2.0 share a core agent engine, bidirectional settings sync, and session export to keep your workflow consistent across both surfaces."
---

Antigravity CLI and the Antigravity 2.0 GUI are built on the same foundation. They share a core agent engine, keep your preferences in sync automatically, and let you move an active terminal session into the GUI when you need a richer environment. The result is a consistent experience across both surfaces — with no duplication of setup.

## Shared agent engine

The CLI and the GUI run on the same core agent engine. Improvements to the engine — including reasoning, multi-file editing, and tool calling — apply to both interfaces automatically.

## Bidirectional settings sync

Preferences and permissions you configure in the CLI are synced to Antigravity 2.0, and vice versa. You only need to set an option once. See [Configure Antigravity CLI settings](/configuration/settings) for details on which preferences are available.

## Session export

You can export a terminal session to the Antigravity 2.0 GUI to continue working in a visual environment. For a full walkthrough, see [Export a session to the GUI](/guides/export-to-gui).
